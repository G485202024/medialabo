「Webコンテンツ作成」の成果

本テーマについて

情報工学基礎知識を応用し実践的な能力を修得することが目的.

到達目標は

  1. 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できる.
  2. コミュニケーション能力の重要性を理解し、グループワークを円滑に実施できる.
  3. モーション解析・編集ができる.
  4. HTML等を用いたWebページ作成ができる.

各授業回の学習事項

第1回 WWW と HTML

HTMLの基本的な構成と見出しやリストなどの要素,特殊文字や空白の説明. HTMLバリデーションでの構文エラーの見つけ方と解決方法. HTML要素リファレンスによる調べ方. 画像やリンクの埋め込み方.

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSSによる Webページの装飾

CSSによる装飾の仕方を学んだ. 外部スタイルシートを使用して，別のファイルから装飾する方法が最も簡単である. 書式にはセクレタとプロパティ名，プロパティ値が必要である. 色の変更だけでなく，回転させたり，レイアウトを変えたり，使い方次第で見栄えの良いWebページを制作できるようになる.

演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

JavaScriptのオブジェクトについて学習した． Javaとは違う方法である変数の宣言，型変換，文字列を囲うダブルとシングルのコーテーション区別が無いこと，のif文の等式や，メソットの作り方． また，ウェブ開発ツールにあるコンソールの詳しい使い方と，デバックの仕方を知った．
